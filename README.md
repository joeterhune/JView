# JView
Judicial Viewer for 12th Judicial Circuit built using ASP.NET Core.  The architecture is based on [Clean Architecture](https://youtu.be/_lwCVE_XgqI)

### Packages and Frameworks used
* Dapper
* FluentValidation
* MediatR
* AutoMapper
* Vue

## Technologies
* .Net Core 2.2
* ASP.NET Core 2.2
